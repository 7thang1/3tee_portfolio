# SRE & System Administrator Portfolio

Welcome to the personal portfolio of **Thi Tan Thinh (3tee)**, a System Administrator and Platform Engineer currently working at Sacombank.

## 🚀 Overview

This is a static, terminal-inspired, single-page portfolio website designed for professionals in DevOps, Site Reliability Engineering (SRE), and System Administration. It features a modern, dark-themed UI with glassmorphism effects and interactive elements.

## ✨ Features

- **Terminal Aesthetic:** Custom "typing" terminal effect simulating CLI commands and system outputs.
- **Particle Background:** Interactive, lightweight canvas-based particle network representing system nodes/connections.
- **Scroll Reveal Animations:** Smooth, scroll-triggered animations for all sections.
- **Responsive Design:** Fully mobile-friendly layout with a customized mobile navigation menu.
- **Tech Stack visualization:** Clean UI for displaying Operating Systems, Containers, and Cloud Platform proficiencies.
- **Project Showcase:** Status-indicator style cards for open source and enterprise projects.

## 🛠️ Built With

- **HTML5:** Semantic structure
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and animations
- **Vanilla JavaScript:** DOM manipulation, IntersectionObserver for animations, and Canvas API for the background
